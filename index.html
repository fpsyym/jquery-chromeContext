chromeContext

Description

Create Chrome-styled context menus (right-click menus) for any element.

Example usage

$('#trigger').chromeContext({
  items: [
    {
      title: 'Hello',
      onclick: function () { console.log('hello.'); }
    },
    { separator: true },
    {
      title: 'World',
      onclick: function () { console.log('world.'); }
    }
  ]
});
